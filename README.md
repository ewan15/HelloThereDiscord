# HelloThereDiscord

## Setup

``` python -m venv .env ```
``` source .env/bin/activate ```
``` pip install discord.py ```
``` pip install python-dotenv ```
``` pip install PyNaCl ```

You need to create a .env file in root as show in sample
```
# .env
DISCORD_TOKEN=<token>
DISCORD_GUILD=<guild_id>
```

## Usage

python main.py
